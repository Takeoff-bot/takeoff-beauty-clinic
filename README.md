# Takeoff Beauty Clinic Website

A modern, responsive website for Takeoff Beauty Clinic with an integrated AI chatbot assistant.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **AI Chatbot Assistant**: Intelligent chatbot that can answer questions about:
  - Services and treatments
  - Pricing information
  - Appointment booking
  - Skincare recommendations
  - Location and hours
  
- **Service Sections**:
  - Facial Treatments
  - Injectable Treatments (Botox, Fillers)
  - Laser & Light Therapy
  - Body Contouring
  - Medical Aesthetics
  - Advanced Skincare

- **Online Appointment Booking**: Easy-to-use appointment request form
- **Provider Profiles**: Meet our expert team
- **Multiple Locations**: Information about all clinic locations
- **News & Blog**: Latest beauty trends and clinic updates

## How to Use

1. Open `index.html` in your web browser
2. Browse through the services and information
3. Click "Book Appointment" to schedule a consultation
4. Chat with the AI assistant (bottom right) for instant answers

## AI Chatbot Features

The AI assistant includes:
- Context-aware responses
- Treatment recommendations
- Pricing information
- Sentiment analysis
- Intent detection
- Quick question buttons
- Personalized interactions

## Technologies Used

- HTML5
- CSS3 (with custom properties and animations)
- Vanilla JavaScript
- Google Fonts (Inter)
- Responsive Grid Layout
- SVG Icons

## Customization

You can customize:
- Colors in `styles.css` (CSS variables in `:root`)
- Services in `index.html`
- AI responses in `script.js` (AI_RESPONSES object)
- Provider information
- Contact details
